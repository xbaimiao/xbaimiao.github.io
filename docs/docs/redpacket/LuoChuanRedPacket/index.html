介绍

前言​

（LuoChuanRedPacket）是一款我的世界红包插件

这个插件是我之前服务器 (洛川) 使用的 测试过的版本为 1.18.2-1.20.1 主要功能特色是支持跨服红包

支持PAPER PURPUR 等paper下游的核心

插件需要链接Redis与MySQ

特色​

一般 金币/点券 红包

口令 金币/点券 红包

发送红包 动画

全 自定义 的配置

简洁的 命令

支持全服同步(即所有子服务器都可查看/领取红包)

命令​

插件权限​

领取与发送红包: luochuanredpacket.command

发送金币红包: luochuanredpacket.command.vault

发送点券红包: luochuanredpacket.command.points

发送口令金币红包: luochuanredpacket.command.vault.text

发送口令点券红包: luochuanredpacket.command.points.text

插件配置​

配置动画​

点击下载 丢进ia的contents文件夹 animatedtitles\configs\particles.yml

里面的贴图改成你自己的

贴图位置animatedtitles\resourcepack\assets\animatedtitles\textures\particles

引自 https://www.mcbbs.net/thread-1464431-1-1.html

配置文件​

# 基本命令权限（玩家领取红包需要这个权限: luochuanredpacket.command
# 发送金币红包: luochuanredpacket.command.vault
# 发送点券红包: luochuanredpacket.command.points
# 口令金币红包: luochuanredpacket.command.vault.text
# 口令点券红包 luochuanredpacket.command.points.text

redis:
  host: localhost
  port: 6379
#  password: ""
mysql:
  enable: false
  host: 'localhost'
  port: 3306
  database: 'minecraft'
  user: 'minecraft'
  passwd: 'minecraft'
  ssl: false

# 金币红包最大金额
maxMoney: 1000
# 点券红包最大金额
maxPoint: 1000
# 不死图腾动画
animation:
  vault: 'server:redpacket'
  point: 'server:redpacket'
# 口令红包黑名单
blacklist:
  # *开头为包含
  - '*傻逼'
  # 全词匹配
  - '你妈死了'
